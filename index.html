☁️ Conectar Google Drive
Configura una sola vez. Después sincroniza solo.
1. Ve a console.cloud.google.com
2. Crea un proyecto
3. Habilita "Google Drive API"
4. Pantalla de consentimiento OAuth → Externo → publica
5. Credenciales → ID de cliente OAuth → App web
6. En "Orígenes de JavaScript autorizados" pon la URL de esta app (ej: https://tuapp.netlify.app)
7. Copia el Client ID abajo
